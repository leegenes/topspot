# topspot
Spotify playlist creator for your top tracks in [three time periods](https://developer.spotify.com/web-api/get-users-top-artists-and-tracks/) powered by [spotipy](http://spotipy.readthedocs.io/en/latest/#)


in addition to [spotipy required environment variables](https://spotipy.readthedocs.io/en/latest/#authorization-code-flow), you'll need to set your Spotify username as a username, too
`export SPOTIFY_USERNAME='acoolusername'`
